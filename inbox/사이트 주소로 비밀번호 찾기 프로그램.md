**Password Finder Program**

Created: 2026-03-28T13:43:48.493312Z

## Problem Statement

The problem is to create a program that can find passwords for given website addresses. The program should be able to handle a large number of inputs and outputs efficiently.

### Input Format

The input consists of two integers N and M, followed by N lines of website addresses and their corresponding passwords, separated by spaces. Each website address is a string consisting of lowercase letters, uppercase letters, dashes (-), and periods (.). The password for each website is also a string consisting only of uppercase letters.

### Output Format

The output consists of M lines, where each line contains the password for the corresponding website address.
